# Analysis Service

## Các bước cài đặt môi trường

### Bước 1: Cài đặt python 3.7.0 (64 bit). Khi cài nhớ click vào 'Add Python 3.7 to PATH'
### Bước 2: Mở Command Prompt
```
pip install pipenv
```

### Bước 3: Đứng tại thư mục của analysis_service
```
pipenv install
```

### Bước 4: Run server
```
pipenv run python app.py
```
